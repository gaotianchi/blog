
| 符号     | 全称                   | 说明      |
| ------ | -------------------- | ------- |
| UR     | user requirements    | 用户需求    |
| RC     | requirements changes | 需求变更    |
| ADD    |                      | 变更方式/添加 |
| REMOVE |                      | 变更方式/移除 |
| UPDATE |                      | 变更方式/更新 |
| E      | Entity               | 实体模型    |

`RC-034 ADD UP-023 备注消息……`

上述字段的含义是：发生需求变动，需求变动编号为 RC-034，变动方式为“添加需求”，添加的是编号为 UR-023 的需求，备注消息为“备注消息……“。
